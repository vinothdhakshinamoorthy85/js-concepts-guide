call / apply / bind

call / apply / bind

Three ways to manually control what this points to

call(), apply(), and bind() all let you set this manually for any function. call and apply invoke the function immediately. bind returns a new function with this permanently locked.
Example 1

call - Invoke Immediately

Set this and pass arguments one by one

function introduce(city, state) {
  console.log("Hi I'm " + this.name + ' from ' + city + ', ' + state);
}
const user = { name: 'Vinoth' };
introduce.call(user, 'Minneapolis', 'MN');
Output
What happened?

call() immediately invokes introduce with this set to user. Arguments are passed one by one after the context object.

Example 2

call - Method Borrowing

Borrow a method from one object to use on another

const person = { name: 'Vinoth', sayhi() { console.log('Hi I am ' + this.name); } };
const other = { name: 'Alice' };
person.sayhi.call(other);
person.sayhi.call(person);
Output
What happened?

call() borrows sayhi and runs it in the context of other. No need to duplicate the method.

Example 3

apply - Args as Array

Same as call but arguments passed as an array

function introduce(city, state) {
  console.log("Hi I'm " + this.name + ' from ' + city + ', ' + state);
}
const user = { name: 'Vinoth' };
const args = ['Minneapolis', 'MN'];
introduce.apply(user, args);
const numbers = [5, 2, 9, 1, 7];
console.log('Max: ' + Math.max.apply(null, numbers));
Output
What happened?

apply() accepts arguments as an array — useful when you already have an array of args.

Example 4

bind - Permanent this

bind returns a new function with this set permanently

function greet(city, country) { console.log(this.name + ' from ' + city + ', ' + country); }
const user = { name: 'Vinoth' };
const bound = greet.bind(user);
bound('Minneapolis', 'USA');
Output
What happened?

bind returns a new function where this is locked to user — call it later or pass it around safely.
